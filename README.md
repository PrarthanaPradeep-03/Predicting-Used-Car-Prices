# Predicting Used Car Prices

## Project Overview
This project predicts the price of used cars using historical data and machine learning techniques. It helps buyers and sellers estimate fair market value based on key vehicle attributes.

## Objective
- Predict used car prices accurately
- Analyze factors affecting car value
- Build a reliable regression model
- Support pricing decisions

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Dataset
Used car dataset containing features like brand, model, year, fuel type, transmission, mileage, and price.

## Project Workflow
1. Data Cleaning & Preprocessing
   - Handled missing values and duplicates
   - Encoded categorical variables
   - Scaled numerical features

2. Exploratory Data Analysis (EDA)
   - Analyzed price distribution
   - Identified key influencing factors
   - Visualized correlations

3. Feature Engineering
   - Selected important features
   - Removed irrelevant variables

4. Model Building
   - Applied regression algorithms (Linear Regression, etc.)
   - Split data into training and testing sets

5. Model Evaluation
   - Evaluated using RMSE, MAE, and R² score

## Key Insights
- Price strongly influenced by car age and mileage
- Brand and fuel type impact resale value
- Proper preprocessing improves model accuracy
